Projects

Yak Language Learner

A Spanish-learning app created with React and Redux. Users input a list of newline-separated Spanish words, and the app dynamically generates flashcards with definitions, examples, and lexical information. From there, users can select the definitions and examples they find most helpful to appear on their flashcards.

Dime

A personal finance app created with React Native, Expo, D3, Firebase, Google Vision API, and Keras.js. Users can input spend by taking a picture of their receipts, scanning barcodes, and manually inputting purchases. Dime takes this data and uses machine learning to predict future spend.
